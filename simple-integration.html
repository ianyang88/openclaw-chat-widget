📖 妙笔助手集成指南

妙笔助手集成时，只需在 URL 中添加 id 参数即可： 

<iframe src="http://8.148.244.33:8080/simple-integration.html?id=用户ID"></iframe>

或者：

<a href="http://8.148.244.33:8080/simple-integration.html?id=用户ID">打开聊天</a>

用户隔离说明：

  • 每个用户 ID 会生成独立的 Session Key
  • 不同用户的聊天历史完全隔离
  • 用户 ID 建议使用唯一标识符（如：工号、用户名等）
  • Session Key 格式：agent:main:direct:url:{userId}

示例： 

用户 A: simple-integration.html?id=zhangsan
用户 B: simple-integration.html?id=lisi
用户 C: simple-integration.html?id=12345
