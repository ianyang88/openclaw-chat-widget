OpenClaw Chat Widget

支持多种认证方式的 OpenClaw 聊天组件

认证模式

配置说明

ws://your-openclaw-host:18789
agent:main:main
Token / Password
OAuth2 with PKCE
agent:main:direct:{userId}
启用 (localStorage)

agent:main:main

使用示例

// Token 认证 (传统方式)
const chatWidget = new OpenClawChatWidget({
    container: '#chat-widget',
    gatewayUrl: 'ws://localhost:18789',
    token: 'your-token-here',
    sessionKey: 'agent:main:main'
});

// OAuth2 认证 (企业SSO)
const chatWidget = new OpenClawChatWidget({
    container: '#chat-widget',
    gatewayUrl: 'ws://localhost:18789',
    auth: {
        type: 'oauth2',
        oauth2: {
            authorizationEndpoint: 'https://accounts.example.com/authorize',
            tokenEndpoint: 'https://accounts.example.com/token',
            clientId: 'your-client-id',
            redirectUri: window.location.origin + '/oauth-callback',
            scope: 'openid profile email'
        }
    },
    session: {
        persist: true,
        autoRestore: true
    }
});

// 监听认证事件
chatWidget.on('authenticated', (data) => {
    console.log('用户已登录:', data.userId);
});

chatWidget.on('logout', () => {
    console.log('用户已登出');
});
